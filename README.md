# Christian · Portfolio V2

Welcome to Version 2 of my personal portfolio. This iteration focuses on a highly refined, constraint-based layout with a minimalist edge, built to be lightning-fast, accessible, and responsive across all devices.

## 🚀 Overview

This portfolio serves as the central hub for showcasing my selected projects, technical expertise, and contact information. V2 was redesigned from the ground up to prioritize:
- **Clarity & Craft:** A clean, distraction-free typography system using custom fonts.
- **Fluid Responsiveness:** A mobile-first approach that seamlessly scales up to a constrained, highly readable desktop layout (max-width 840px).
- **Performance:** Built with vanilla web technologies to ensure a perfect lighthouse score and instant load times.

## ✨ Features

- 🌓 **Dynamic Theme System:** Built-in Light/Dark mode toggle that persists user preference via `localStorage`.
- 📱 **Mobile-First Layout:** Optimized stacking for smaller viewports with specialized CSS Grid rules for larger screens (e.g., 2x2 project cards on desktop).
- 🧩 **Iconography:** Integrated pixel-perfect SVG icons driven by Iconify.

## 🛠 Proposed Tech Stack

- **REACT:**
- **HTML5:** Semantic markup structure.
- **CSS3:** Custom properties (variables), Flexbox, and CSS Grid (Zero CSS frameworks used).
- **JavaScript (Vanilla):** DOM manipulation for theme switching and interactive elements.
- **Iconography:** [Iconify](https://iconify.design/) + Font Awesome.

## 🖥 Running Locally

Because this project is built entirely with static files, there's no build step required. 

1. Clone the repository:
   ```bash
   git clone https://github.com/christian-fx/PortfolioV2.git
   ```
2. Navigate to the project directory:
   ```bash
   cd PortfolioV2
   ```
3. Open `index.html` in your favorite browser, or serve it using a local development server (like VS Code Live Server).
